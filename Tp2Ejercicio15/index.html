HOLA MUNDO ESTE ES UNA PRUEBA PARA SABER QUE SELECTOR TIENE MAS PRIORIDAD:

SI EL TEXTO ES DE COLOR NARANJA, EL SELECTOR CON MAS PRIORIDAD ES EL SELECTOR DE ELEMENTO

SI EL TEXTO ES DE COLOR AZUL, EL SELECTOR CON MAS PRIORIDAD ES EL SELECTOR DE CLASE

SI EL TEXTO ES DE COLOR VIOLETA, EL SELECTOR CON MAS PRIORIDAD ES EL SELECTOR DE ID
